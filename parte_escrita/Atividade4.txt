4) Explique como você utilizaria estruturas if para controlar o sistema
Utilizaria o seguinte pseudocódigo para exemplificar.

Iniciar
  definir led_rgb verde
  ler botao1
  ler botao2
  ler botao3
  ler potenciometro1
  mapear potenciometro1 em 0 a 255 
#definindo os modos dos botões
  Se botao1 = 1:
   definir modo = 1
  
  Se botao2 = 1:
    definir modo = 2
	
  Se botao3 = 1:
  	definir modo = 3

 Se modo == 0:
	 definir led_rgb verde

  Se modo == 1:
	  definir led_rgb amarelo

  Se modo == 2:
	 definir led_rgb vermelho

  Se modo == 3:
	 definir led_rgb baixo
	 aguardar 500 milesimos
	 definir modo == 0
Fim

Explicação:
Nesse sentido, podemos dizer que em cada função, os 3 primeiros IFs estarão ligados no quesito de definir os botões, 
Nos outros 3 IFs estarão sendo relacionados com a mudança das cores (verde para amarelo/amarelo para vermelho),
o IF relacionado ao botão 3 servirá para apagar o LED RGB e em 0,5 segundos (500 milésimos), retornar a luz verde, 
no sentido de reiniciar o progresso.
