4) Explique como você utilizaria estruturas if para controlar o sistema
Utilizaria o seguinte pseudocódigo para exemplificar.

Iniciar
  definir led_rgb verde
  ler botao1
  ler botao2
  ler botao3
  ler potenciometro1
  mapear potenciometro1 em 0 a 255 

  Se botao1 = 1:
    esperar 2 segundos
    definir led_rgb amarelo

  Se botao2 = 1:
	  esperar 2 segundos
	  definir led_rgb vermelho

  Se botao3 = 1:
	  definir led_rgb baixo
	  esperar 5 segundos
	  definir led_rgb verde
Fim
	
Nesse sentido, podemos dizer que em cada função, os 2 primeiros IFs estarão ligados no quesito de mudança 
de cor (verde para amarelo/verde para vermelho/amarelo para vermelho), caso seja necessário para 
reiniciar o sistema, o IF relacionado ao botão 3 servirá para apagar o LED RGB e em 5 segundos, 
retornar a luz verde, no sentido de reiniciar o progresso.
